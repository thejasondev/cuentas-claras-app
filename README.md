# Cuentas Claras

Aplicación para dividir la cuenta de restaurante entre comensales. Calcula el total por persona incluyendo propina.

## Despliegue en Vercel

1. Haz fork o clona este proyecto.
2. Sube el repositorio a Vercel. Vercel detectará Next.js automáticamente.
3. El logotipo oficial de la app está en `/public/Cuentas Claras Logo.png`; este se usa como favicon e imagen principal.

### Variables de entorno
Copia `.env.example` a `.env.local` y modifica lo necesario.

### Branding & Logo
- Imagen pública oficial: `/public/Cuentas Claras Logo.png`. Úsala donde se necesite branding, favicon, redes sociales, etc.
- Si generas builds customizadas mantén la marca en todas partes visibles.
